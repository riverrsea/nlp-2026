==================== 第三阶段任务要求 ====================

1. 统一评估指标
所有模型都需要输出 Accuracy、Macro Precision、Macro Recall、Macro F1、每个类别的 Precision、Recall、F1、classification_report、confusion_matrix。请将 Macro F1 作为重点指标。

2. 训练并评估 TF-IDF + SVM
在 train.csv 上训练；在 val.csv 上调参或选择参数；在 test.csv 上评估；保存预测结果和评估结果。

3. 训练并评估 TextCNN-Random
使用 CrossEntropyLoss；使用 Adam；默认 learning_rate = 1e-3；默认 batch_size = 32 或 64；默认 epoch = 10；保存最佳模型；记录 train loss、val loss、val accuracy、val macro-F1；保存训练曲线。

4. 训练并评估 TextCNN-Pretrained
与 TextCNN-Random 使用完全相同的超参数；只改变 Embedding 初始化方式；保存最佳模型；记录 train loss、val loss、val accuracy、val macro-F1；保存训练曲线；在 test.csv 上评估；与 TextCNN-Random 做性能对比。

5. 训练并评估 BERT Fine-tuning
使用 AdamW；默认 learning_rate = 2e-5；默认 batch_size = 8 或 16；默认 epoch = 3；默认 max_length = 256；在 val.csv 上保存 macro-F1 最高的模型；在 test.csv 上评估；保存训练曲线。

6. 运行 Prompt Learning 实验
至少支持 BERT Prompt Zero-shot、BERT Prompt Few-shot、GPT Prompt Zero-shot 接口、GPT Prompt Few-shot 接口。Prompt 实验也要在 test.csv 上评估。

7. 生成最终模型对比表

8. 绘制训练曲线

9. 绘制混淆矩阵
对表现最好的模型绘制混淆矩阵。保存为 outputs/figures/confusion_matrix.png。横轴为预测类别，纵轴为真实类别。类别名清晰可读，可以使用中文类别名或英文类别名，但要保持一致。

10. Case Study 错误案例分析
从最佳模型在 test.csv 上的错误预测中挑选 2 到 3 个典型样本。每个样本需要包含：原始文本片段、真实标签、预测标签、错误原因分析。保存为 outputs/results/case_study.md。

11. 最终 README
README.md 需要包含：项目简介、数据集格式说明、三个阶段说明、工程框架说明、cfg 配置文件说明、main.py 参数说明、如何运行数据准备、如何训练 TF-IDF + SVM、如何训练 TextCNN-Random、如何训练 TextCNN-Pretrained、如何训练 BERT、如何运行 Prompt Learning、输出文件说明、如何查看最终模型对比结果、常见问题，例如 HuggingFace 模型无法下载、预训练词向量路径缺失等。

==================== 建议运行顺序 ====================
请在 README.md 中写明推荐运行顺序：
1. 数据准备：bash scripts/run_data_prepare.sh 或 python src/data_prepare.py
2. 训练传统模型：bash scripts/run_svm.sh 或 python src/main.py --config cfg/svm.yaml --mode train --output_dir outputs
3. 训练 TextCNN-Random：bash scripts/run_textcnn_random.sh 或 python src/main.py --config cfg/textcnn_random.yaml --mode train --output_dir outputs
4. 训练 TextCNN-Pretrained：bash scripts/run_textcnn_pretrained.sh 或 python src/main.py --config cfg/textcnn_pretrained.yaml --mode train --output_dir outputs --pretrained_path path/to/vector.txt
5. 训练 BERT：bash scripts/run_bert.sh 或 python src/main.py --config cfg/bert.yaml --mode train --model_name_or_path bert-base-chinese --output_dir outputs
6. 运行 BERT Prompt：bash scripts/run_prompt_bert.sh
7. 运行 GPT Prompt：bash scripts/run_prompt_gpt.sh
8. 只测试某个模型：bash scripts/test_model.sh 或 python src/main.py --config cfg/textcnn_random.yaml --mode test --checkpoint outputs/checkpoints/textcnn_random_best.pt --output_dir outputs
9. 查看最终结果：outputs/results/all_model_comparison.csv
